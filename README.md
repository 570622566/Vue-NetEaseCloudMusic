# Vue-NetEaseCloudMusic

基于vue 2.0  模仿IOS版网易云音乐的手机网站，
在启动这个项目之前，你需要在本地跑一个nodeAPI服务
技术栈  vue,vuex,vue-router ,axios,es6
完成功能个性推荐，歌单，主播电台列表，排行榜，
听歌，上一首，下一首。
